# Airbnb Data Analytics Case Study

## Project Overview

This project analyzes Airbnb listing data to uncover insights into pricing, room types, host behavior, guest reviews, and property availability. The analysis was performed using SQL for data exploration and Tableau for interactive data visualization.

The objective is to transform raw Airbnb data into meaningful business insights that can help hosts, customers, and business stakeholders make informed decisions.

---

## Dataset

**Dataset:** Airbnb Open Data

The dataset contains information about Airbnb listings, including:

- Listing ID
- Host information
- Neighbourhood Group
- Neighbourhood
- Room Type
- Price
- Service Fee
- Availability
- Number of Reviews
- Review Ratings
- Minimum Nights
- Instant Booking
- Cancellation Policy
- Construction Year

## Project Workflow

1. Imported Airbnb dataset into MySQL.

- Room Type Pricing
- Review Rating Analysis

---

## Key Business Insights

- Average listing prices vary significantly across neighbourhoods.
- Entire homes/apartments generally have higher prices than private or shared rooms.
- Guest review ratings help identify high-performing listings.
- Availability patterns reveal seasonal and booking trends.
- Room type and location strongly influence listing prices.
